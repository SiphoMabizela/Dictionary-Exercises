# Dictionary-Exercises
Python Exercises on data handling, JSON making data persistent, saving &amp; reload data, CSV Files (Structured Data), CSV to dictionary

Exercise 1: Build a Student Profile
Create a dictionary representing a coding bootcamp student:
Must include:
  name (string)
  email (string)
  course (string)
  completed_projects (list of strings)
  is_active (boolean)

Then:
1. Update the course name
2. Add a new project to the list
3. Remove the is_active key


Excercise 2: Mini Booking System
Build a simple calendar system:
1. Create an empty dictionary called calendar
2. Ask user for a date
3. Create an event dictionary with volunteer name and status
4. Store the event inside calendar under the date key
5. Print the full calendar

Bonus:
Change an event's status from 'Available' to 'Booked'
